# 黃金期貨進出場策略 — 高維度分析與優化 (2026-07)

## 目標
用高維度視野分析黃金期貨 (GC=F) 順勢突破策略, **提高風險報酬比、降低最大回撤 (DD)**,
且不破壞既有的抗過擬合紀律。

## 起點: 現行生產策略
`gold_strategies.py` — 小時線 24H 收盤突破進場、2.5×ATR(14) 移動停損出場、只做多、固定 1 口。

| 指標 | 基準值 |
|---|---|
| 交易數 | 168 |
| 勝率 | 48.8% |
| PF | 1.93 |
| 總損益 | +$201,610 |
| **最大DD** | **$22,290** |
| **報酬/回撤比** | **9.04** |
| Sharpe (逐筆) | 0.223 |
| 逐年 | 2024 +41k / 2025 +105k / 2026 +56k |

核心 3 參數 (breakout=24 / atr_stop=2.5 / atr_n=14) 已由 walk-forward + 樣本外 +
日線26年『四關』驗證為穩健最優 (見 `gold_strategies.py` docstring), **本次不再碰**。

## 方法: 只掃『正交』新維度, 雙關卡驗證
「高維度」= 不動進出場**價格規則**, 改動另外三軸: **下多少單 / 何時允許進場 / 如何了結**。
每個維度都跑兩道關卡:
1. **小時線** — 訓練 (2024-25, IS) / 樣本外 (2026, OOS) 分離, OOS 不得惡化;
2. **日線 26 年** — 跨 regime (含 2013-15 空頭、2020 崩跌、2021-22 盤整), 這是決定性的第 4 關。

工具: `gold_hd_study.py` (四維總掃)、`gold_gate_validate.py` (趨勢閘細掃+日線關)、
`gold_dd_levers.py` (冷卻+ATR天花板, 內建日線關)。

## 結果: 六個維度逐一檢驗

| 維度 | 小時線表現 | 日線26年 | 裁決 |
|---|---|---|---|
| **D1 波動率目標部位** | DD 分文未降 ($22,290 不動), OOS 凍結在 +56k | — | ✗ 只是把 2024 低價低波動段放大槓桿的假象 |
| **D2 慢速趨勢閘 (MA)** | MA125/150 漂亮 (DD↓16.6k, PF↑2.1) 但屬**單點尖峰**, 鄰居不確認 | **全面惡化** (DD 23k→28-29k), 空頭年虧損未縮 (2015 -17k 原封不動) | ✗ 突破本含動能, 閘 redundant; 與既有『MA no-op』結論一致 |
| **D3 分段了結 (+kR)** | k 愈大愈近基準, 提早分批只是砍利 (讓獲利奔跑最好) | — | ✗ 弱 |
| **D4 ADX 趨勢閘** | 提 PF 但砍掉 OOS 獲利 (56k→48k) | — | ✗ |
| **L1 停損後冷卻** | 單調惡化 (剔除的多是有效再進場) | 單調惡化 | ✗ 乾淨否決 |
| **L2 ATR 天花板** | 2.5× 從不觸發 (=基準); 收緊反傷 | 弱base上像降DD, 但**採用配置上只砍利、DD不降** (見下方追加專項) | ✗ 對弱基準的相對改善 = 過擬合 |

### 關鍵洞察
- **D1 波動率目標的陷阱**: 在單一離散口數 + 金價由 $2000 漲到 $4000 的樣本裡, 高價段口數
  被地板卡在 1 口, vol-target 只放大了早期低價段, DD 完全沒降、OOS 零變化 → 是槓桿假象不是降DD。
- **D2 趨勢閘的陷阱**: 小時線的漂亮結果是 2024-26 單一多頭 regime 的產物, 一放到日線 26 年就翻車
  —— 正是本專案 docstring 一再警告的過擬合模式。突破訊號本身已隱含價格站上短中期均線,
  再加 MA 閘幾乎 redundant, 攔不到『站在均線上方才崩』的空頭反彈假突破。
- **L2 ATR 天花板看似真信號, 實為過擬合**: 初看它在日線降 DD (濾崩跌型爆量假突破), 但那個
  增益只在『次優 base』成立, 換到日線採用配置就蒸發 —— 詳見下方追加專項的 base 敏感度檢驗。

## 結論與落地
**核心策略已在效率前緣**; 六個正交維度**無一**通過完整檢驗 (含 base 敏感度) →
**生產參數維持不變**, 這本身確認並延伸了既有紀律 (把「不要再加濾網」用證據回答清楚)。

`atr_ceiling` 機制以**預設關閉的研究開關**形式收錄 (非採用):
- 進場當根 ATR > `atr_ceiling` × 近 200 根 ATR 中位數 → 否決該筆進場 (爆量假突破)。
- **預設 `None` = 關閉**, 現行小時線績效**逐筆完全不變** (168筆/PF1.93/DD$22,290/9.04, 已驗證 byte-identical)。
- 經 `gold_ceiling_robust.py` 過擬合複驗**否決** (見下方追加專項): 增益隨 base 強度消失。
  保留純供日後複現/再驗, 對現行績效零影響。

## 追加專項: ATR 天花板到底有沒有 overfit? (gold_ceiling_robust.py) — 結論: **有, 否決**

針對「把 L2 融入系統是否有更佳績效、是否過擬合」做五角度硬檢驗。**前四關看似都過, 第五關翻盤。**

**(1)~(3) 初看像真 edge** (但注意: 這三關都用【小時參數 bo24/atr2.5 硬套日線】的弱 base):
- (1) 2D 平台: `ceiling ∈ [1.25,1.5] × 窗任意` 一致把該 base 的 DD $30k→$20k, 像真平台。
- (2) 逐事件: 擋掉 10 筆、避開淨虧 −$38,700, 分散在 8 個獨立年份, 不像單一事件。
- (3) 前後半: 兩個子期都改善或持平。

**(4) 小時線 (生產時間軸)** — 日線最優的 1.5× 在小時線**反而傷** (DD $22k→$30k、總損益 −$16k);
唯一不傷的 2.5× 是 no-op。已是警訊: 效益不跨時間軸。

**(5) base 配置敏感度 (決定性, 翻盤)** — 把同一個 1.5× 天花板換到日線**實際採用配置**
(bo40/atr4/ATR20, 本就用寬停損+長突破避開洗損), 增益整個蒸發:

| base | 無天花板 | +ceil 1.5× | 差異 |
|---|---|---|---|
| 弱 base bo24/atr2.5 (前四關用的) | DD $27.7k / 總 $162.7k | DD $20.4k / 總 $198.6k | **ΔDD −7.3k / Δ總 +35.9k** ✅ 漂亮 |
| **採用配置 bo40/atr4** (日線實跑) | DD $23.0k / 總 $271.8k | DD $23.0k / 總 $218.7k | **ΔDD 0 / Δ總 −53.1k** ❌ 只砍利 |

**增益隨 base 強度消失 = 這是『相對於一個被刻意削弱的基準』的改善, 不是絕對 edge = 過擬合。**
天花板在弱 base 上避掉的那些洗損, 在採用配置裡早就被『寬 ATR 停損 + 長突破窗』處理掉了;
再加天花板只是把好棒也一起濾掉、砍掉利潤。

### 對兩個提問的直接回答
- **「有 overfit 嗎?」** → **有**。表面的日線增益是拿次優 base 量出來的假象; 一放到真正的
  採用配置就歸零甚至反傷。前四關會過, 正是因為它們都共用同一個弱 base —— 這是很隱蔽的
  過擬合形態 (對基準的相對改善), 靠第 (5) 關的 base 敏感度才抓得到。
- **「融入系統有更佳績效嗎?」** → **沒有**。
  - 小時線『生產』策略: 1.5× 傷績效、2.5× no-op → 無增益。
  - 日線策略: 在實際採用配置上只砍利、DD 不降 → 無增益。

### 結論 (據此決定落地)
- **生產小時線與日線一律維持 `atr_ceiling=None` (關閉)** —— 誠實不硬塞一個假改善。
- `atr_ceiling` 參數**保留為預設關閉的研究開關** (`gold_strategies` / `gold_daily_backtest`),
  純供日後複現/再驗, 對現行績效**零影響** (小時線逐筆 byte-identical)。
- **方法學教訓**: 評估任何濾網/增強前, 先確認 base 已是各自時間軸的合理最優; 拿弱 base
  當對照會系統性高估增益。這條已併入 `gold_ceiling_robust.py` 的第 (5) 關作為常駐檢查。
