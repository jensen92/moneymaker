# MoneyMaker — 台股飆股每日篩選系統

每日從台灣上市股票(成交值前 500 檔)中,以兩個策略各篩出最多 2 檔候選股,
並以 200 萬本金、最高 2 倍槓桿進行歷史回測驗證。

## 流程

```bash
pip install pandas numpy requests
python3 download_data.py   # 下載 3 年日線 (TWSE OpenAPI 清單 + Yahoo Finance 歷史價)
python3 backtest.py --leverage 2   # 回測兩策略
python3 daily_picks.py     # 輸出今日訊號 → 明日進場候選
```

## 策略

**共同條件**:股價 > 10 元、日成交值 > 5 千萬、大盤濾網(加權指數收盤 > 60 日均線才開新倉)。

### 策略 A「強勢拉回」— 目標高勝率、風險報酬比 2
- 多頭排列(close > MA20 > MA60,MA20/MA60 皆上揚)
- 近 20 日漲幅 10–50%(動能股但不過熱)
- 拉回觸及 MA20 後當日收回 MA20 之上
- 停損 2.5 ATR,停利 2R,最長持有 20 日
- 近 20 日漲幅條件為 15–60%(經 `sweep_a.py` 參數掃描選定)

### 策略 B「爆量突破」— 目標風險報酬比 ~5、勝率 3 成以上
- 收盤突破前 60 日高點
- 成交量 ≥ 2.5 倍 20 日均量
- 收在當日 K 棒上緣 25% 內、近 20 日漲幅 ≤ 60%(避開噴出末端)
- 初始停損 1.5 ATR,Chandelier 移動停利(最高收盤 − 3 ATR),最長持有 60 日

## 回測假設
- 進場:訊號日次一交易日開盤價
- 成本:手續費 0.1425% 雙邊、證交稅 0.3%、滑價 0.1% 單邊
- 部位:單筆風險 = 權益 1%,整股(1000 股)為單位,總名目曝險 ≤ 權益 × 槓桿
- 每策略每日最多新進 2 檔

## 檔案
| 檔案 | 說明 |
|---|---|
| `download_data.py` | 下載股票清單與日線資料至 `data/` |
| `strategies.py` | 兩個策略的訊號邏輯與技術指標 |
| `backtest.py` | 組合層級回測引擎 |
| `daily_picks.py` | 每日選股輸出(含停損價與建議股數) |
| `sweep_a.py` | 策略 A 參數掃描工具 |

## 回測結果 (約 3 年, 2023-06 ~ 2026-06, 2 倍槓桿)

| 策略 | 交易數 | 勝率 | 獲利因子 | 總報酬 | 年化 | 最大回撤 |
|---|---|---|---|---|---|---|
| A 強勢拉回 | 346 | 43.4% | 1.16 | +21.2% | +6.7% | 33.5% |
| B 爆量突破 | 583 | 23.5% | 1.22 | +87.4% | +23.4% | 43.8% |

策略 B 在 1 倍槓桿下因曝險上限限制持倉數,報酬大幅下降 (+4%),
其優勢來自廣撒網讓少數大贏家(移動停利常跑出 5R 以上)拉動整體。

## 風險提醒
回測結果不代表未來績效;槓桿會放大虧損。實單前建議先以小部位驗證,
並注意 Yahoo 資料未含除權息調整造成的訊號誤差。
